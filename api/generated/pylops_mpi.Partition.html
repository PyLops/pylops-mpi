pylops_mpi.Partition#

class pylops_mpi.Partition(value)[source]#

Enum class

Distributing data among different processes.

  • BROADCAST: Distributes data to all processes (ensuring that data is kept consistent across processes)

  • UNSAFE_BROADCAST: Distributes data to all processes (without ensuring that data is kept consistent across processes)

  • SCATTER: Distributes unique portions to each process.
